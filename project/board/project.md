# CloudDeploy

## Description
A CLI tool for deploying containerized applications to multiple cloud providers (AWS ECS, GCP Cloud Run, Azure Container Apps). Features: multi-provider abstraction, environment management (dev/staging/prod), rollback support, health checks, deployment logs, and a web dashboard for monitoring. Tech stack: Python 3.12, Click CLI, httpx for API calls, SQLite for local state, Rich for terminal UI.
